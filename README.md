<h1 align="center">Ping</h1>
<p align="center">Aplicación de mensajería instantánea para dispositivos Android.</p>

## Tecnologias y herramientas utilizadas

Ping utiliza diferentes tecnologías y herramientas, incluyendo:

- Firebase para el manejo de la autenticación, almacenamiento y notificaciones en tiempo real
- Facebook Authentication para el inicio de sesión mediante cuentas de Facebook
- Google Authentication para el inicio de sesión mediante cuentas de Google
- Inicio de sesión con email y contraseña
- SharedPreferences para el almacenamiento local de datos
- Coroutines para la gestión de hilos de ejecución
- ViewModel y LiveData para el manejo del ciclo de vida de los componentes
- Dagger Hilt para la inyección de dependencias
- Retrofit para la gestión de las solicitudes HTTP
- Arquitectura Clean Architecture para la organización y estructuración del código

## Funcionalidades

- Inicio de sesión con varias opciones: La app permite a los usuarios iniciar sesión a través de sus cuentas de Google, Facebook o mediante el ingreso de su correo electrónico y contraseña.
- Agregación de descripción del perfil: Una vez que los usuarios han iniciado sesión, pueden agregar una descripción a su perfil para compartir información sobre sí mismos con otros usuarios de la app.
- Mensajería en tiempo real: La app utiliza Firebase para proporcionar a los usuarios una funcionalidad de chat en tiempo real, lo que les permite enviar y recibir mensajes instantáneos.
- Mensajes recientes: Además de la funcionalidad de chat en tiempo real, la app también muestra una lista de los mensajes más recientes para que los usuarios puedan ver fácilmente las conversaciones que han tenido en el pasado.
